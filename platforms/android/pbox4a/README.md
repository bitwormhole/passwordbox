# pbox4a

pbox4a (PasswordBox for Android) 是 PasswordBox 的 Android 客户端 App.

目前, pbox4a 项目基于这些 SDK 开发:

| SDK            | 版本                 | 备注 |
| -------------- | -------------------- | ---- |
| Java (JDK)     | 11                   |      |
| Android  (SDK) | Android 8  (API 26) |      |
| Android  (NDK) |                      |      |
