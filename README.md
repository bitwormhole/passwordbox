# PasswordBox

PasswordBox 是一个极简的生成式密码管理器。

本 App 的核心部分使用 go 实现,  并使用 Java (对于 Android) 或 Swift (对于 iOS) 开发其用户界面。

## 功能特性

1. 完全开源 (非常重要! 作为密码管理器,这类应用必须完全开源, 不能把密码放在未知的黑盒子里面);
1. 使用现代密码学算法加密 (安全性);
1. 原生移动平台应用 (便捷性);
1. 使用基于 hash 算法的链式数据块存储机密材料 (工作原理类似 git);
1. 优先使用强密码生成算法, 适合大部分使用场景;
1. 支持自定义密码加密存储算法, 兼顾小部分特殊的使用场景;
1. 使用云端侧链, 防止在客户端暴力破解, 并提供对密码解锁操作的审计功能;
1. 支持把生密码 (Raw-Password) 加工成熟密码 (Cooked-Password), 使用熟密码更安全;
1. 尽可能避免添加第三方依赖, 减少通过依赖项引入的不可知或不可控的安全风险;
1. 支持通过 IME 向本设备其它应用输入密码(可选);
1. 支持通过 BLE-HID 设备(基于esp32s3), 向其它设备输入密码(可选);
